Projects and Assignments

I am currently undertaking the John Hopkins University Data Science Certification composed of ten courses. Courses are assessed through peer-reviewed assignments. Here are some of the assignments and projects I've done. Follow the links for more details:

  • Prediction Models: The project used a weight lifting excercise dataset to build three different models (a simple classification tree, a boosted predictor using the boosted trees method and a random forest predictor) in order to predict the manner (how well) in which they did the exercise.
  • Statistical Inference: This project is reported in two parts. Part 1 is a simulation excercise to investigate the exponential distribution in R and compare it with the Central Limit Theorem. Part 2 demonstrates basic inferential data analysis by exploring the ToothGrowth data in the R datasets package.
  • Regression Models: This project applied regression to answer “Is Manual or Automatic transmission more fuel efficient?”. My report explored the relationship between a set of variables and miles per gallon, with least squares and inference using regression of linear and multivariable models.
  • Reproducible research: This project involved analysis of the NOAA storm event database. It required processing a large and messy dataset to determine the most harmful extreme weather events with respect to population health and economics across the United States. I used the knitr package in R to publish my analysis with my code embedded in the document.
  • Exploratory Data Plots: Explored the "Individual household electric power consumption Data Set" taken from the UC Irvine Machine Learning Repository . Required loading, converting and subsetting the data by date and time before generating the desired plots.
  • Getting and Cleaning Data: This project demonstrates my ability to collect, work with, and clean a data set. The dataset recorded six human activities by accelerometers from the Samsung Galaxy S smartphone.
  • Missing Values: This assignment tested my ability to write functions that interact with different files of data and to deal with data containing missing values. I was required to write three different functions extracting and processing data from different files contained in a given directory. The first function 'pollutantmean.R' calculates means. The second function 'complete.R' determines the number of complete cases, i.e. where the data in each file is complete and has no missing values. The third function 'corr.R' calculates a correlation between different data based on a threshold value of complete cases.
  • Cashing a matrix :A simple R function to help save time - as well as learning to fork, commit and push changes to github
